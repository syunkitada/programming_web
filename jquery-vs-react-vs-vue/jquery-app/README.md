# Getting Started with Create React App with JQuery

- JQuery で UI 生成をごり押すパターン
- UI ライブラリ: Twitter Bootstrap
- グラフ: Google Chats

```
$ yarn create react-app jquery-app --template typescript

$ yarn add @types/google.visualization
$ yarn add @types/jquery


public/index.html に以下を追加する
```
